## 项目结构

* app [后台业务]
 * action [控制层]
 * common [公共]
  * config [配置文件]
  * helper [操作类库]
 * controllers [装饰层]
 * model [数据模型]
 * routes [路由表]
 * utils [工具]
 * views [模版 - 暂时没用到]
* bin [运维]
  * logs [日志]
  * wapman.sql [数据库脚本]
  * www [启动脚本]
* middleware [中间件]
  * connect [koa混合中间件]
  * database [数据连接中间件]
  * root [根目录中间件 - 星空背景]
  * router [路由绑定中间件]
  * sess [会话中间件]
* public [静态目录]
